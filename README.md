# Linear Algebra Library in C++

A step-by-step recreation of a linear algebra library in C++, inspired by Lyndon Duong’s project.  
This repository is focused on **learning by building**, with an emphasis on understanding core concepts in both **linear algebra** and **C++ system design**.

---

## 📌 Purpose

The goal of this project is not just to implement linear algebra operations, but to:

- Deeply understand how matrix and vector operations work under the hood
- Strengthen C++ fundamentals (memory, classes, templates, performance)
- Bridge mathematical theory with real-world implementation
- Practice writing clean, modular, and testable code

---

## 🧠 Concepts Covered

This project gradually builds toward a full linear algebra system by implementing:

### Core Structures
- Vectors
- Matrices

### Operations
- Vector addition & scalar multiplication
- Dot product
- Matrix multiplication
- Transpose
- Determinant
- Inverse (planned / in progress)

### Advanced Topics (Planned)
- LU Decomposition
- Eigenvalues / Eigenvectors
- Numerical stability considerations

---

## ⚙️ Tech Stack

- **Language:** C++
- **Compiler:** g++ / clang++
- **Build System:** Makefile
